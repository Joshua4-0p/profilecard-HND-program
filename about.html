Bio

I'm a budding frontend engineer who enjoys building simple, accessible UI components and learning modern patterns. I focus on semantic HTML, responsive CSS, and pragmatic JavaScript.

Goals in this program

  • Sharpen semantic and accessible HTML skills.
  • Master responsive UI patterns with CSS Grid and Flexbox.
  • Practice writing testable, maintainable JavaScript.

Areas of low confidence

  • Advanced CSS animations and performant patterns.
  • Complex accessibility patterns (e.g., ARIA-complex widgets).
  • State management for larger vanilla JS apps.

Note to future self

Keep practicing, ask for feedback early, and try to break components intentionally to learn failure modes. Shipping small, consistent improvements helps a lot.

Extra thoughts

A strong habit: make every component accessible by default and progressively enhance visuals. Focus on clarity over cleverness.
